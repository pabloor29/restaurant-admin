07 — Motion

Transitions & Animation

Animations subtiles, fonctionnelles. Transitions courtes (150ms) pour les états hover/focus. Transitions moyennes (300–500ms) pour les entrées/sorties de vues.

Token Durée Easing Usage
transition-fast 150ms ease Couleur border au focus, opacity hover, changement de couleur de texte
transition-base 200ms ease Boutons hover, badges, background switches
sheet-close 300ms ease-in-out Fermeture drawer/sheet latéral (Radix Sheet)
sheet-open 500ms ease-in-out Ouverture drawer/sheet latéral
fade-in-0 200ms ease Apparition overlays, modals, tooltips
zoom-in-95 200ms ease Scale entrée dialog/popover (0.95 → 1)
slide-in-from-* 300ms ease-out Slides directionnels (drawer, toast)
Principes
Fonctionnel
Aucune animation décorative. Chaque transition guide l'attention ou confirme une action.
Discret
Durées courtes (150–300ms). L'interface ne doit jamais "bloquer" l'utilisateur pour une animation.
Cohérent
Mêmes durées pour les mêmes types d'actions. ease-in-out pour les contenants, ease pour les états.
